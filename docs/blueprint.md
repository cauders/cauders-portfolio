# **App Name**: Cauders Portfolio

## Core Features:

- Homepage Showcase: Display a curated selection of featured projects on the homepage to highlight key works.
- Filterable Portfolio: Enable users to filter projects by category (e.g., 'Web App', 'Mobile App', 'AI/ML') for easy browsing.
- Project Detail Pages: Create dynamic project pages showcasing project details, galleries, technologies used, and descriptions.
- Contact Form: Implement a simple contact form for inquiries.
- Dynamic Content Generation: Use a tool that suggests text, style changes and graphic design options appropriate to each portfolio item.
- Shared Components: Develop shared Header and Footer components for consistent navigation across all pages.

## Style Guidelines:

- Primary Action Color: Modern cyan/teal (#8CEAE5 in light mode, #C2F2EE in dark mode) for interactive elements.
- Background Color: Very light off-white/gray (#F8F8F8) in light mode, deep dark blue (#1D293A) in dark mode.
- Foreground Color: Dark muted blue (#2C3E50) in light mode, light off-white (#F0F8FF) in dark mode for text.
- Headline Font: 'Montserrat' (sans-serif) for all main headings. Note: currently only Google Fonts are supported.
- Body Font: Default sans-serif for body text. This matches the request.
- Clean layouts with plenty of white space; card-based layouts for portfolio items.
- Subtle fade-in-on-scroll effects for content, hover effects for cards (flip or scale), and button color transitions.
